# Takuzu 
A Takuzu game implementation in Pharo.

# How to install

To install the core of the game and the unit tests:
```
Metacello new
    baseline:'Takuzu';
    repository: 'github://Enzo-Demeulenaere/Takuzu:master/repository';
    load: 'core'
```
